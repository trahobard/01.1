# Настройка окружения

---

Если вы вдруг все еще не поставили Git или не зарегистрировались на GitHub, то явно что-то не так &ndash;
большой повод срочно этим заняться.

## 1. Установка Git и GitHub

1. Установите систему контроля версий [Git](https://git-scm.com/downloads)
    * Можете использовать графические оболочки для гита, если вам удобно
    * В частности, можете работать с гитом из интерфейса PyCharm
2. Зарегистрируйтесь (если еще не сделали этого) на [GitHub](https://github.com)

**Следующий шаг &ndash; для работы с Git в терминале**.
Вам скорее всего это не понадобится, если вы не планируете глубоко погружаться в гит
и использовать его из консоли/терминала.

<details>
<summary>Настройка SSH-ключа</summary>
3. Создайте SSH-ключ для доступа к своим репозиториям
    * Воспользуйтесь Git Bash, который должен был установиться вместе с Git, если вы используете Windows
    * Вам может
      помочь [эта страница](https://docs.github.com/en/github/authenticating-to-github/connecting-to-github-with-ssh/about-ssh)
    * Еще одну понятную инструкцию по настройке можно найти [здесь](https://sendel.ru/posts/https-to-ssh-on-github/) 
    * Никогда не давайте никому свой приватный ключ!

Альтернативный вариант, чтобы не генерировать ключ &ndash; подключить PyCharm (см. далее) напрямую к
GitHub. Или склонировать репозиторий с помощью HTTPS.
</details>

## 2. Установка языка и среды

1. Установите Python не ниже 3.8, лучше &ndash; последнюю стабильную версию
   на [официальном сайте](https://www.python.org/downloads/)
    * Убедитесь, что `python` или `python3` можно вызвать из командной строки, для этого при установке желательно
      выбрать все возможные галочки
    * Если все еще не работает ни `python`, ни `python3`, попробуйте `py`
    * На всякий случай обновите пакетный менеджер `pip` (для этого придется запустить скрипт от имени администратора) с
      помощью команды `python -m pip install --upgrade pip`
2. Установите среду программирования [PyCharm](https://www.jetbrains.com/pycharm/)
    * Выбирайте бесплатную Community версию
